# Сайт для GitHub Pages

Готовый статический сайт: HTML, CSS, JavaScript и локальная картинка в `assets/`.

## Как опубликовать

1. Создайте репозиторий на GitHub с названием `ваш-логин.github.io`.
2. Загрузите в него все файлы из этой папки.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Через пару минут сайт будет доступен по адресу `https://ваш-логин.github.io/`.

## Что поменять под себя

- В `index.html`: имя, описание, проекты, контакты.
- В `styles.css`: цвета, размеры, отступы.
- В `assets/hero.png`: главную картинку, если захотите другую.
